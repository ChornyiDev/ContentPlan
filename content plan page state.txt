contentPlanFullDoc  (Document)                 документ завантажений з бази та з   бережений в стейт для подальших дій
campaignsFullList   (List<Document>)           список кампаній завантажений з бази цього документу та збережений в стейт для подальших дій. відображений на сторінці як кнопковий фільтр
adsFullList         (List<Document>)           список реклами завантажений з бази цього документу та збережений в стейт для подальших дій. це не фільтрований список він буде використовуватися для подальшої фільтрації
tagList             (List<Document>)           список тегів завантажений з бази цього документу та збережений в стейт для подальших дій. відображений на сторінці як кнопковий фільтр
filterByCampaignRef (DocumentReference)        Посилання на кампанію за якою фільтрується список реклами. записується при натисканні на кнопку кампанії.
filterByStatuses    (List<String>)             Список статусів за якими фільтрується список реклами. записується при натисканні на кнопку статусу. 
filterByPlatforms   (List<String>)             Список платформ за якими фільтрується список реклами. записується при натисканні на кнопку платформи.
tempTagToEditTagFilter (String)                додатковий стейт який використовується при застосуванні фільтру в інтерфейсі. для бекенду не важливий
filterByTags        (List<DataType>)           Список тегів за якими фільтрується список реклами. Записується при завантаженні сторінки але з пустим масивом тегів. теги в масив записуються по натисканні на теги в інтерфейсі. приклад тегу з пустим масивом: {"category": "Audience", "options": [] }
filterByTagSimple   (List<String>)             Додатковив масив тегів для фільтрації у спрощеному вигляді. записується при натисканні на теги в інтерфейсі.

    "options": ["B2B"]
